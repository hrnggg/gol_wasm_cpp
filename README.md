# wasm+cpp game of life

## Requirements
- [Emscripten](https://emscripten.org/index.html)
- C++17
- ES2015(ES6)

## Usage
### setup
```
source </path/to/emsdk>/emsdk_env.sh
```

### build & run
```
make build
```

### run
```
make run
```

### format js/html/css
```
npm install
npm run format .
```
